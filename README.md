## Screens Included
- Home Screen (article list)
- Article Detail Screen  
- Saved Articles Screen

## Interactions
- Click "Read More" → navigates to Article Screen
- Click Back button → returns to Home
- Click Bookmark icon → saves article
- Click Saved icon → opens Saved Articles

## Animations
- Slide transitions between screens 

## User Testing Feedback
Tested with 2 users.

**User 1:** Found navigation smooth. Suggested bookmark feedback.

**User 2:** Said the design was clean and easy to use. Recommended "Saved!" confirmation.
